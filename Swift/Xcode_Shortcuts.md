##  Navigator Panel
```
⌘ + 0 = Show or Hide the Navigator Panel
⌘ + 1 = Project
⌘ + 2 = Symbol
⌘ + 3 = Find
⌘ + 4 = Issue
⌘ + 5 = Tests
⌘ + 6 = Breakpoint
⌘ + 7 = Report
```

## Utility Area
```
⌥ + ⌘ + 0 = Show or Hide the Utility Area
⌥ + ⌘ + 1 = File Inspector
⌥ + ⌘ + 2 = Quick Help Inspector
⌥ + ⌘ + 3 = Identity Inspector
⌥ + ⌘ + 4 = Attributes Inspector
⌥ + ⌘ + 5 = Size Inspector
⌥ + ⌘ + 6 = Connections Inspector
```
